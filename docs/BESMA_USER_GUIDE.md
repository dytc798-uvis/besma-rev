# BESMA 사용설명서

이 문서는 BESMA 주요 메뉴를 빠르게 이해하고, 현장/본사 사용자가 바로 실행할 수 있도록 정리한 안내서입니다.

## 시작하기

- 로그인 후 좌측 사이드바에서 메뉴를 선택합니다.
- 본사(HQ)는 설정/관리 중심, 현장(SITE)은 실행/업로드 중심으로 사용합니다.
- 메뉴가 많을 경우 `사용설명서` 메뉴에서 기능별 설명을 바로 확인하세요.

## 대시보드

- 오늘 처리해야 할 항목과 현재 상태를 한눈에 확인합니다.
- 현장(SITE)은 실행 우선, 본사(HQ)는 모니터링 우선으로 구성됩니다.

## 공지사항

- 공지 등록/조회/댓글로 전달사항을 공유합니다.
- SITE 화면 상단에는 최신 공지 제목이 티커로 표시됩니다.

## 안전보건 방침 및 목표

- HQ는 본사 방침/목표 파일을 업로드하고 관리합니다.
- SITE는 기본적으로 현장 방침/목표를 보고, 필요 시 본사 내용을 전환 조회합니다.

## 안전 교육

- 교육자료를 목록으로 확인합니다.
- HQ 권한으로 교육자료 업로드가 가능합니다.

## 안전 점검

- 문서취합의 안전점검(INSPECTION) 파일을 게시판 형태로 확인합니다.
- 항목별 댓글로 점검 관련 소통을 할 수 있습니다.

## 부적합사항

- 부적합사항관리대장(xlsx) 업로드 후 항목이 파싱되어 표시됩니다.
- 원본 다운로드, PDF 보기, 개선조치(내용/일자/담당자/사진) 편집이 가능합니다.

## 근로자의견청취

- 관리대장 업로드(xlsx/csv) 후 의견 항목을 자동 파싱합니다.
- 생년월일/휴대전화/의견종류(아차사고/유해위험요인발굴) 지원.
- 3단계 워크플로우: `현장승인 -> 본사확인 -> 포상후보`.

## 문서 탐색

- PDF 파일 중심으로 문서를 검색/열람합니다.
- 기준자료와 현장 문서를 통합 탐색할 수 있습니다.

## 내 현장 문서 / 문서 취합 현황

- SITE는 업로드/재업로드/상태확인을 수행합니다.
- HQ는 전체 현장 상태 모니터링 및 검토를 수행합니다.

## 소통자료

- SITE 현장 간 소통용 자료를 공유하고 미확인 건수를 확인합니다.

## 운영 아이디어 제안

- 운영 개선 아이디어를 등록하고 이력을 확인합니다.

## 설정

- 현장 정보/설정 값을 확인하거나 변경합니다.
- HQ의 문서 설정관리에서는 문서 항목/주기/동적 메뉴를 관리합니다.

## 동적 메뉴(본사 설정)

- HQ 설정에서 게시판형/표형 메뉴를 추가할 수 있습니다.
- 메뉴 순서는 드래그앤드롭 후 `메뉴 순서 저장`을 눌러 확정됩니다.
- 동적 메뉴는 방침 및 목표 아래에 노출됩니다.

## 구글설문 연동 운영(A안)

- 구글폼 응답을 CSV로 내려받아 `근로자의견청취`에 업로드합니다.
- 자동 로그인 방식(아이디/비밀번호 저장)은 권장하지 않습니다.
- 운영 권장 순서:
  1) 구글폼 응답 수집
  2) CSV 다운로드
  3) BESMA 업로드 및 검토

## 스크린샷 촬영 가이드

- 사용설명서 페이지는 메뉴별 스크린샷 카드를 표시합니다.
- 이미지가 없는 경우 "스크린샷 준비중" 안내가 자동 표시됩니다.
- 실제 운영 화면 캡처는 `frontend/public/guide-shots/README.md` 파일명 규칙을 따라 배치하면 됩니다.
