# 현장 문서관리 사용방법 (간단)

이 문서는 현장 사용자가 오늘 바로 써야 하는 기능만 1장으로 정리한 요약 가이드입니다.

## 오늘부터 바뀐 점

- 문서취합은 `Requirement 기반`으로 정리되어 현재 회차와 재조치 필요 문서를 구분해서 봅니다.
- `수정 업로드`는 반려된 기존 회차에 다시 연결되어 이력이 이어집니다.
- `근로자의견청취`, `부적합사항`은 파일 업로드가 기본이고, 직접 입력은 `수동 row 추가 (보조)`를 펼칠 때만 사용합니다.
- HQ 승인/반려와 별도로 문서별 `코멘트`를 남길 수 있습니다.
- 날씨는 실시간이 아니라 `KST 05:00 / 12:00` 스냅샷 기준으로 보이며, 실패 시에도 안전 문구로 종료됩니다.

## 현장에서 할 일 3가지

### 1. 문서 업로드

- `내 현장 문서`에서 오늘 제출할 항목을 확인합니다.
- 필요한 파일을 올리고 `보기`로 바로 확인합니다.

### 2. 수정 업로드

- `재조치 필요 문서`에서 반려된 항목을 찾습니다.
- `문서 수정 업로드`로 다시 제출하면 기존 이력에 이어서 저장됩니다.

### 3. 의견청취 / 부적합사항 등록

- 기본은 관리대장 파일 업로드입니다.
- 예외적으로 직접 입력할 때만 `수동 row 추가 (보조)`를 펼쳐 사용합니다.

## 캡처 1. 문서취합

`[캡처 삽입 예정]`

- 권장 화면: `현장 문서취합 (Requirement 기반)` 첫 화면
- 함께 보여줄 요소: 할 일 목록, 문서 업로드 버튼, 재조치 필요 문서 영역

## 캡처 2. 근로자의견청취

`[캡처 삽입 예정]`

- 권장 화면: 파일 업로드 영역 + `수동 row 추가 (보조)` 접힘 상태
- 함께 보여줄 요소: 업로드 버튼, 목록, 상태칩

## 캡처 3. 부적합사항

`[캡처 삽입 예정]`

- 권장 화면: 관리대장 업로드 영역 + `수동 row 추가 (보조)` 접힘 상태
- 함께 보여줄 요소: 업로드 버튼, 목록, PDF/개선조치 버튼

## 한 줄 정리

- 현장은 `업로드`, `수정 업로드`, `관리대장 등록` 3가지만 익히면 기본 운영이 가능합니다.
